# Data Cleaning & Exploratory Data Analysis (SQL)

## Project Overview
This project focuses on cleaning and exploring raw data using SQL to improve data quality and prepare it for analysis and reporting.

## Tools
- SQL

## Data Cleaning Steps
- Removed duplicate records to ensure data uniqueness  
- Standardized data formats for consistency  
- Handled null and blank values based on logical rules  
- Removed unnecessary columns to optimize the dataset  

## Exploratory Data Analysis
- Analyzed data structure and distributions  
- Identified patterns, trends, and potential anomalies  
- Validated data accuracy after cleaning  

## Outcome
The final dataset is clean, standardized, and ready for dashboards, reporting, and data-driven decision-making.
